\doxysubsection{Detailed Description}
Класс \doxylink{class_task}{Task} представляет задачу в менеджере задач. 

Хранит описание, срок выполнения, приоритет, категорию, теги и статус выполнения. Позволяет управлять этими параметрами через методы класса.
